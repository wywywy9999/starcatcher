# StarCatcher（揽星）

告别收藏夹吃灰，打造你的专属个人知识库。

StarCatcher 是一款开源的智能书签与个人知识库工具，帮你统一管理零散的网页、笔记、复盘与备忘内容。

你是否也有过这样的困扰：刷到一篇好文随手收藏，转头就再也找不到？收藏夹里堆满了链接，却从来没有再打开过？

StarCatcher 帮你解决这些问题：

- 粘贴链接，AI 自动抓取内容、生成摘要、分类打标，让你一眼看懂内容，再也不用打开原网页回忆
- 支持三种记录模式，不管是想偷懒用 AI，还是想手动整理，或是纯粹写备忘，都能满足
- 所有数据都存在你的本地电脑，隐私安全有保障，不用担心数据泄露
- 灵活的筛选、排序、导出功能，让你的知识体系井井有条

> 目前仅支持 Windows 系统。

---

## 三种记录模式，适配不同场景

不管你是想快速收藏网页，还是想仔细整理笔记，都有适合你的模式：

| 模式 | 适用场景 | 功能说明 |
|---|---|---|
| AI 自动模式 | 快速收藏好文，不想手动整理 | 粘贴链接后，自动抓取网页内容，通过 DeepSeek 生成摘要、分类与标签，一键完成收藏 |
| 手动填写模式 | 想自己定义内容，保留原链接 | 粘贴链接后，手动填写标题与摘要，完全自定义内容，适合需要精准整理的场景 |
| 纯笔记模式 | 记录个人想法、备忘 | 无需链接，直接写标题、摘要与正文，就是一个纯粹的本地备忘录工具 |

## 核心功能

- **卡片式管理**：直观的书签卡片网格，支持按分类、标签、关键词快速筛选，找内容一秒到位
- **AI 智能处理**：可选的 DeepSeek AI 能力，自动生成摘要、分类与标签，解放你的双手
- **灵活分类**：支持多分类、多标签体系，拖拽即可调整排序，轻松整理你的知识体系
- **批量操作**：支持批量添加内容，也能一键导出为 Markdown / CSV 格式，数据随时可迁移
- **正文批注**：在自己添加的备忘录中，选中文字即可添加批注，右侧侧边栏统一管理，重点内容一目了然
- **个性化管理**：支持置顶重要内容、随时编辑修改、添加个人笔记，让内容更贴合你的需求

## 为什么选择 StarCatcher？

- **轻量本地部署**：无需服务器，一键启动即可使用，所有数据都存在你的本地 SQLite 数据库，隐私完全由你掌控
- **AI 可选，灵活适配**：AI 功能完全可选，没有 API Key 也能正常使用所有基础功能，当普通书签 / 备忘录工具完全没问题
- **兼顾收藏与笔记**：不止是网页收藏，还能记录你的个人笔记、复盘、备忘，一个工具搞定所有知识管理需求
- **开源免费**：完全开源免费，你可以自由修改、定制，适配自己的使用习惯

---

## 技术栈

- **前端**：Next.js + Tailwind CSS + React Query，现代化的前端架构，流畅的交互体验
- **后端**：Python FastAPI + SQLAlchemy + SQLite，轻量高效，无需额外的数据库服务
- **AI 能力**：DeepSeek API（可选），强大的大模型能力，帮你自动处理内容

## 快速开始

### 环境准备

确保你的本地已经安装了 Python 3.8+ 和 Node.js 18+。目前仅支持 Windows 系统。

### 一键启动（Windows）

Windows 用户可以直接双击项目根目录的 `start.bat`，即可一键启动前后端服务。

### 手动启动

如果你想手动启动，可以按照以下步骤操作：

1. 克隆项目到本地

```bash
git clone https://github.com/wywywy9999/starcatcher.git
cd starcatcher
```

2. 启动后端服务

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

3. 启动前端服务

```bash
cd frontend
npm install
npx next dev --port 3000
```

4. 打开浏览器，访问 `http://localhost:3000` 即可开始使用

---

### 配置 AI 功能（可选）

如果你想使用 AI 自动摘要、分类功能，需要配置 DeepSeek API Key：

1. 创建 `backend/.env` 文件（可参考 `backend/.env.example`）

```env
DEEPSEEK_API_KEY=你的DeepSeek API Key
DEEPSEEK_BASE_URL=https://api.deepseek.com
DATABASE_URL=sqlite:///./data/linkvault.db
```

2. 将 `你的DeepSeek API Key` 替换为你的真实 Key

> 提示：你可以在 [DeepSeek 官网](https://platform.deepseek.com/) 申请 API Key。

## 个人使用场景

StarCatcher 非常灵活，你可以用它来做很多事情：

### 面试准备

我个人主要用它来管理面试相关的内容：

- 面试复盘笔记，记录每次面试的问题与收获
- 行为问题回答思路，整理自己的经历，方便快速回忆

### 其他场景

除了面试，你还可以用它：

- 收藏技术文章，AI 自动总结重点，方便之后复习
- 记录学习笔记，整理课程内容与自己的思考
- 记录项目灵感，随时记下想到的创意
- 日常备忘，管理待办、想法等零散内容
